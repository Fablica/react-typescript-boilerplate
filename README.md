## Setup

```
$ npm install
```

or

```
$ docker/up
```

## Scripts

```
$ npm run start:dev
$ npm run build
$ npm run build:dev
$ npm run lint
$ npm run fmt
$ npm run test
$ npm run test:watch
$ npm run test:feature
$ npm run storybook
$ npm run build-storybook
```

or

```
$ docker/npm run start:dev
$ docker/npm bash
$ docker/exec whoami # root
```
